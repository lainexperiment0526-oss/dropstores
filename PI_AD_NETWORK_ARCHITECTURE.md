## ğŸ“Š Performance Considerations

### Session Storage
- **Stored**: Ad count, last ad timestamp
- **Size**: ~50 bytes
- **Cleared**: On tab close
- **Reset**: Manual via `resetSession()`

### Memory Usage
- **Hook**: Minimal state (3 values)
- **Components**: Lightweight, no heavy dependencies
- **SDK**: Loaded once globally

### Network Requests
- **Ad Ready Check**: ~100ms
- **Ad Request**: ~200-500ms
- **Ad Display**: Handled by Pi SDK
- **Verification**: ~300-500ms (backend)
